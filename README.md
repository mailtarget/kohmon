# Kohmon

Collection of kotlin common helper


### Usage

To use in gradle project
    
    dependencies {
        implementation "co.mtarget:kohmon-core:0.0.8"
    }
    

#### Kohmon Core

this module contains simple helpers for strings, regex, coroutine and date
